# Lattice Documentation

This directory is prepared for GitHub Pages. Use [Documentation Home](index.md)
as the entry point once Pages is enabled.

Generated API docs are checked in under [api/latest/](api/latest/index.html)
after running `./gradlew docsJavadoc`. GitHub's repository browser shows these
as plain HTML files; they render as normal Javadocs only when served by GitHub
Pages or another static host.

The root repository page remains
[README.md](../README.md).
Project policy files are linked from both the root README and the documentation
home page.
